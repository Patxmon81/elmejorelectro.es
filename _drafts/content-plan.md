## Formato de los borradores

- Archivo: `_drafts/YYYY-MM-DD-slug.html`
- Seguir exactamente la estructura de `/pages/review-philips-hd9650.html`
- Incluir: ficha técnica, análisis, pros/cons, puntuación, CTA Amazon

## Instrucciones para publicar un borrador

Cuando el usuario aprueba un borrador:
1. Mover el archivo de `_drafts/` a `pages/`
2. Añadir la URL al `sitemap.xml`
3. Añadir la review a la página `pages/reviews.html`
4. Commit y push a GitHub
